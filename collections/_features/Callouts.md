Callouts are an important part of the typing experience, where different callouts serve different purposes.


## Input & Action Callouts

Input callouts are used to highlight the currently pressed key, while action callouts are used to show secondary actions when a key is long pressed.

## Namespace

KeyboardKit has a ``Callout`` namespace with callout-related types and views, a ``CalloutContext``, auto-persisted ``CalloutSettings``, callout customizations, etc.
